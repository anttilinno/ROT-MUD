    2. A package-level string-interning registry (D-05) — no exact in-repo analog; design from the decisions. Also expose a read-only `func lookupCapability(key string) (int, bool)` that returns the bit if already registered (no allocation), used by the zero-alloc query path in Plan 02.
    Keep the registry's bit-assignment counter and map at package scope; document the determinism + 256 ceiling invariants in doc comments.
