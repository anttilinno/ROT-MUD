Pre-commit hook enforcement is deferred (RESEARCH Pitfall #6 recommends a later phase); this plan relies on commit-message visibility and PR review to catch silent updates. |
| T-01-03-02 | Repudiation | Parity-gate bypass (developer `-update`s to make CI green) | mitigate | CI must run `go test ./...` without `-update` (VALIDATION.md Sampling Rate; also RESEARCH Assumption A2 and `.mise.toml` task `test: "go test ./..."` verified). | T-01-03-03 | Information disclosure | Snapshot file contains sensitive values | accept | The fixture constructs characters from canned `NewCharacter("Caster"/"Victim"/...)` calls; no player save is loaded; no config.toml read; no env vars read. Threat model verified in Plan 02 T-01-02-03. |
